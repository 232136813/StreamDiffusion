# Txt2Img Example

<p align="center">
  <img src="../../assets/demo_01.gif" width=80%>
</p>

StreamDiffusion を用いた GUI を提供します。
入力プロンプトを変更すると、テキストから 4x4 の画像をリアルタイムに生成することができます。

## 使用方法

以下のコマンドを順番に実行してください。

```bash
pip install -r requirements.txt
cd view
npm i
npm start &
cd ../server
python main.py
```
